## Aurora - Langauge-Learning-Platform
Aurora - Where Every Word Shines Bright
Aurora is an interactive language learning and translation platform designed to make learning fun and efficient. Whether you're brushing up on vocabulary or translating phrases from English to French, Spanish, German, and more, Aurora has you covered. The user interface is intuitive, with interactive features such as voice input, real-time language translation, and a responsive design.

Quick Access Preview : https://aurora-langauge-learning-app.netlify.app/translator

## Features
Real-time Language Translation: Supports translation between multiple languages (English, French, Spanish, German, etc.).
Voice Input and Output: Allows users to input text via speech and listen to translations.
Clean, Responsive UI: Designed for seamless interaction across various devices.
Dropdown Language Selector: Easy-to-use dropdown menus for selecting source and target languages.
Speech Synthesis and Recognition: Uses browser APIs for voice commands and playback.

## Technologies Used
HTML/CSS: For structuring and styling the web pages.
JavaScript: For adding interactivity, handling translations, and voice input/output.
Speech Recognition API: Enables voice input for translations.
Google Translate API: Optional external API for broader translation capabilities (not included in the current version).
Chart.js: Used for visualizations within the learning progress tracker.

## Prerequisites
Basic knowledge of HTML, CSS, and JavaScript.
A modern web browser (Chrome, Firefox, Safari, etc.).
Running Locally
1) Clone the repository:
git clone https://github.com/username/aurora-where-every-word-shines-bright.git

2) Navigate to the project directory:
cd aurora-where-every-word-shines-bright

3) Open the index.html or translator.html file in your browser to get started.

### Usage 
Select a Language Pair: Choose the source and target language from the dropdown menus (e.g., English → French).
Input Text: Type the word/phrase you want to translate into the left text area.
Translate: Click the "Translate" button in the middle to display the translated text on the right.
Voice Input: Click the microphone icon to input text via speech.
Listen to Translation: Click the speaker icon to hear the translated phrase aloud.
